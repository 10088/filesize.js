About

filesize.js is a tiny JavaScript library that creates a Global function named filesize().

It provides an easy way to get a human readable file size String. It works great with all kinds of APIs that describe files.

filesize.js is an AMD module, which can be lazy loaded with AMD loaders like require.js and curl.js.

Examples

filesize() accepts a String or Number (Integer or Float), and an optional second parameter as a Number, which is the decimal place to round to (default is 2).

It returns a String that can be read by people.
