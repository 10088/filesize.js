What does it do?

filesize.js is a tiny JavaScript library that creates a Global function named filesize().

It provides an easy way to get a human readable file size String.

How do I use it?

filesize() accepts a String or Number (Integer or Float), and 2 optional parameters: a Number for the decimal place to round to (default is 2) or true for "short" output, and a second Boolean argument is supported to disable `bit` sizes, e.g. "Kb".

filesize.js can be loaded from npm, as an AMD module, or as a regular script.
